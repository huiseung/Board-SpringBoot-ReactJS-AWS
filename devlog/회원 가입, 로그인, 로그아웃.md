# Backend

## log
- level
  - 로그 출력 레벨을 설정하면 보다 낮은 레벨의 로그는 출력되지 않는다
1) error: 오류 발생
2) warn: 처리가능하나 향후 문제 발생 가능성 있음
3) info: 상태 변경
4) debug: 디버깅
5) trace: 좀더 상세한 디버깅
